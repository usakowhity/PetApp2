🐾 PetApp2 – AIペットアプリシリーズ

🟦 はじめに

あなたの呼びかけ声や笑顔に反応するペットアプリ。
USBやSDカードにあなたの愛したペットの心が宿り、
それを挿しただけで休眠パソコンが癒しのAIペットロボットに変わります。

🟩 エディション一覧

エディション名種類内容ダウンロード

🟦 使い方（共通）

  1. ZIP を展開
  2. run.bat を実行
  3. ペットプロフィールを編集
  4. 外部AIツールで画像・動画を生成
  5. generated/ フォルダに配置
  6. Play モードで再生

🟩 特徴

  • インストール不要（Portable）
  • Python Embedded + ffmpeg 同梱
  • Windows 10/11 対応
  • ペットプロフィールからプロンプト自動生成
  • 外部AIツールとの連携を支援
